Приложение для управления заказами. Для установки надо запустить PythonInstaller.exe, затем installtools.bat. После чего программу всегда можно будет использовать через ManagerOrders.bat. Приложение добавляет заказы, загружая требуемые файлы, создает таблицу онных и выгружает их в нужный файл в файловой системе pickle.
